
## 利用 YouTube 快速轻松提高英语水平


![how to use Ebuoy.jpg](/assets/img/how to use Ebuoy.jpg)

1. 找到喜爱的 YouTube 视频, 一键分享到 Ebuoy.
2. 一边放视频, 一边读字幕文章.
3. 单击查词(配合欧陆字典), 长按标记单词为已掌握/未掌握.
